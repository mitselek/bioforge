# BioForge

Terminal-based ecosystem simulation. Four species (plants, herbivores, carnivores, decomposers) evolve on a torus-topology 2D world under a hard energy-conservation invariant. Each creature's behavior is driven by a small Turing-machine genome that mutates across generations.
